# winston-slack-transport

A Slack transport for Winston 3+ that logs to a channel via webapi and api token.

## Options

| option      | required | default          | description |
| ----------- | :------: | ---------------- | ----------- |
| token       | x        |                  |             |
| channel     | x        |                  |             |
| name        |          | "slackTransport" |             |
| attachments |          |                  |             |
| blocks      |          |                  |             |
| mrkdwn      |          | true             |             |
| text        |          |                  |             |
| level       |          | info             |             |
